📖 Documentation de l'API
1

Générez votre clé API

Cliquez sur "Générer une clé" dans la section ci-dessus. Copiez-la immédiatement — elle ne sera plus visible en clair ensuite.

2

Configurez votre caisse

Dans votre logiciel de caisse, cherchez "Webhook", "Intégration" ou "Export automatique". Renseignez :

  • URL : https://alteore.com/api/public-ca
  • Méthode : POST
  • Header : Authorization: Bearer {votre_clé}
  • Content-Type : application/json
3

Testez avec curl

Remplacez alte_votreclé... par votre vraie clé et testez depuis un terminal :

# Envoi d'un CA journalier multi-TVA curl -X POST https://alteore.com/api/public-ca \ -H "Authorization: Bearer alte_votreclé..." \ -H "Content-Type: application/json" \ -d '{ "date": "2026-03-04", "ht055": 0, "ht10": 150.00, "ht20": 320.50 }' # Réponse attendue { "ok": true, "updated": ["4/3/2026"], "message": "1 jour(s) mis à jour" }
📌 Format 1 — Multi-TVA (recommandé)
{ "date": "2026-03-04", // ISO, FR (04/03/2026), ou DD-MM-YYYY "ht055": 0, // CA HT à 5,5% (alimentaire, livres…) "ht10": 150.00, // CA HT à 10% (restauration, transport…) "ht20": 320.50 // CA HT à 20% (standard) }
📌 Format 2 — Montant unique + taux TVA
{ "date": "2026-03-04", "montantHT": 470.50, // Montant total HT "tvaRate": 20 // 5.5 | 10 | 20 }
📌 Format 3 — Tableau de plusieurs jours
{ "lignes": [ { "date": "2026-03-03", "ht10": 120, "ht20": 280 }, { "date": "2026-03-04", "ht10": 150, "ht20": 320.50 } ] } // Maximum 100 lignes par appel
📌 Format 4 — Additionner au CA existant (mode "add")
{ "date": "2026-03-04", "ht20": 120.00, "mode": "add" // "replace" (défaut) ou "add" } // "add" = cumule sur la valeur existante
